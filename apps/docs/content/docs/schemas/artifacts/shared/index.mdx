---
title: Shared Schemas
description: Shared types used across artifacts
---

Paradoc defines shared types that are used across multiple artifact types.

| Type                                         | Description                                              |
| -------------------------------------------- | -------------------------------------------------------- |
| [Layer](/schemas/artifacts/shared/layer)     | Rendering output specification (inline or file-backed)   |
| [ContentRef](/schemas/artifacts/shared/content-ref) | Inline text or file-backed content reference (instructions, agent instructions) |
| [Defs & Rules](/schemas/artifacts/shared/logic) | Typed computed values and validation rules             |
