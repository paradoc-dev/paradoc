title: Artifact Schemas
description: The four artifact types in Paradoc
---

Paradoc has 4 basic artifacts:

| Artifact                                  | Description                                                                                                                                                                               |
| ----------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [Document](/schemas/artifacts/document)   | Static documents (e.g., disclosures, pamphlets)                                                                                 |
| [Form](/schemas/artifacts/form)           | Dynamic documents that have parties, fields, annexes, and/or logic (e.g., contracts, applications, certificates) |
| [Checklist](/schemas/artifacts/checklist) | Compliance documents that track the completion of tasks or requirements (e.g., closing checklist)                                                               |
| [Bundle](/schemas/artifacts/bundle)       | Containers for Documents, Forms, Checklists and even other Bundles (e.g., loan application packet)                                                                                   |
