All tools available on the Paradoc MCP server.

| Tool | Category | Description |
|------|----------|-------------|
| [`validate`](/mcp/tools/validate) | Core | Validate an artifact against the Paradoc schema and logic rules |
| [`fill`](/mcp/tools/fill) | Core | Fill a form or checklist with data, apply defaults, and validate |
| [`render`](/mcp/tools/render) | Rendering | Render a form from a verified registry as text, PDF, or DOCX |
| [`list_artifacts`](/mcp/tools/list-artifacts) | Registry | Browse artifacts ranked by popularity |
| [`get_artifact`](/mcp/tools/get-artifact) | Registry | Get the full artifact JSON and metadata from a verified registry |
| [`list_registries`](/mcp/tools/list-registries) | Registry | List registries with artifact counts and install stats |
| [`get_registry`](/mcp/tools/get-registry) | Registry | Get details for a specific registry and its artifacts |
| [`search`](/mcp/tools/search) | Registry | Search artifacts and registries by keyword |
