Displays information about Paradoc Manager.

# Usage

```bash
para about
```

# Description

The `about` command shows version and environment information for the Paradoc CLI, including the CLI version, platform, Node.js version, and current working directory.

# Examples

Show about information:

```bash
para about
```

# Output

```
Paradoc Manager

Version:        0.1.0
Platform:       darwin
Node Version:   v20.10.0
Working Dir:    /Users/you/my-project
```
