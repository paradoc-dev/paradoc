Reset the Paradoc CLI to factory defaults.

# Usage

```bash
para reset [options]
```

# Options

| Option | Description |
|--------|-------------|
| `-y, --yes` | Skip confirmation prompt |
| `--keep-registries` | Keep registry configurations |
| `--keep-cache` | Keep cached data |

# Description

Performs a factory reset of the CLI by:

1. Clearing the registry cache (`~/.paradoc/cache`)
2. Removing all installed renderer plugins (`~/.paradoc/renderers`)
3. Creating a fresh default configuration (`~/.paradoc/config.json`)

You will be prompted for confirmation unless `--yes` is passed.

# Examples

Full reset:

```bash
para reset -y
```

Reset but keep registries:

```bash
para reset --keep-registries
```

Reset but keep cache:

```bash
para reset --keep-cache
```
