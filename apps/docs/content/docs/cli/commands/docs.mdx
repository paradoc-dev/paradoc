Opens the Paradoc documentation website in your default browser.

# Usage

```bash
para docs
```

# Description

The `docs` command provides quick access to the Paradoc documentation. When executed, it opens `https://docs.paradoc.dev` in your default web browser.

# Examples

Open documentation:

```bash
para docs
```

Output:

```
Opening Paradoc documentation...
```
