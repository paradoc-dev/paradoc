Opens the Paradoc web console in your default browser.

# Usage

```bash
para console
```

# Description

The `console` command provides quick access to the Paradoc web console at `https://paradoc.dev`. When executed, it opens the URL in your default web browser.

# Examples

Open the web console:

```bash
para console
```

Output:

```
Opening Paradoc console...
```
