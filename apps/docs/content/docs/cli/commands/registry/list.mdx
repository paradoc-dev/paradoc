List all configured registries.

# Usage

```bash
para registry list [options]
para registry ls [options]
```

# Options

| Option | Description |
|--------|-------------|
| `--json` | Output as JSON |

# Output

```
Configured registries (2):

@acme [project]
  https://registry.acme.com

@company [global]
  https://registry.company.com
```
