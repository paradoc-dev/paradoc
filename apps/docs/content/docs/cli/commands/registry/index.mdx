Manages registry configurations for your project and global settings.

# Usage

```bash
para registry <command> [options]
```

# Subcommands

| Command | Description |
|---------|-------------|
| [add](/cli/commands/registry/add) | Add or update a registry |
| [remove](/cli/commands/registry/remove) | Remove a registry |
| [list](/cli/commands/registry/list) | List configured registries |
| [make](/cli/commands/registry/make) | Create a new registry.json file |
| [info](/cli/commands/registry/info) | Show registry information |
| [catalog](/cli/commands/registry/catalog) | Manage registry catalog items |
| [stats](/cli/commands/registry/stats) | Show registry statistics |
| [compile](/cli/commands/registry/compile) | Compile registry artifacts |
| [view](/cli/commands/registry/view) | View details of an installed artifact |
