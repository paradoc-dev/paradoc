All tools available via `paradocTools()` in `@paradoc/ai-sdk` and `@paradoc/tanstack-ai`.

| Tool | Category | Description |
|------|----------|-------------|
| [`validateArtifact`](/ai/tools/validate-artifact) | Core | Validate an artifact against the Paradoc schema from artifact JSON, URL, or registry |
| [`fill`](/ai/tools/fill) | Core | Fill a form or checklist with data from artifact JSON, URL, or registry |
| [`render`](/ai/tools/render) | Rendering | Render a form to PDF, markdown, or DOCX from artifact JSON, URL, or registry |
| [`getRegistry`](/ai/tools/get-registry) | Registry | Fetch registry.json from a URL, returns available artifacts |
| [`getArtifact`](/ai/tools/get-artifact) | Registry | Fetch artifact JSON from a registry by name |
