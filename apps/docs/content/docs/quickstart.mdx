Let's build a simple purchase agreement to grasp the basics of Paradoc. We'll define the agreement, populate it with data, validate it, and render the output.

<Step>
### Define the Form

Let's define a simple purchase agreement with 2 parties and 3 fields. It will have a single layer that is a Markdown file.

```ts
import { para } from '@paradoc/sdk'

const purchaseAgreement = para
  .form()
  .name('purchase-agreement') // required
  .title('Purchase Agreement')
  .version('1.0.0')
  .parties({
    buyer: para.party().label('Buyer').signature({ required: true }),
    seller: para.party().label('Seller').signature({ required: true }),
  })
  .fields({
    quantity: para.field.number().label('Quantity').required(),
    price: para.field.money().label('Price').required(),
    date: para.field.date().label('Date').required(),
  })
  .layers({
    markdown: para.layer().file().mimeType('text/markdown').path('purchase-agreement.md'),
  })
  .defaultLayer('markdown')
  .build()
```
</Step>

In this Quickstart, we're using the fluent builder pattern to define the form. You can also use the object pattern if you prefer.

<Step>
### Create the Layer

Next, let's create the layer. We'll use Handlebars syntax to map the form parties/fields to the layer template.

```md title="purchase-agreement.md"
# Purchase Agreement

{{parties.buyer.name}} agrees to purchase {{quantity}} units at {{price}} each.
{{parties.seller.name}} agrees to deliver by {{date}}.
```

Paradoc is agonstic to the rendering engine and file format you choose. You can use any rendering engine and file format you want.

</Step>

<Step>
### Populate

Let's populate the form with data by using the `.fill()` method.

```ts
const draft = purchaseAgreement.fill({
  fields: {
    quantity: 100,
    price: { amount: 25, currency: 'USD' },
    date: '2025-03-01',
  },
  parties: {
    buyer: { id: 'buyer-1', name: 'Alice Johnson' },
    seller: { id: 'seller-1', name: 'Bob Smith' },
  },
})
```
</Step>

<Step>
### Validate

Next, let's check that the data is valid by using the `.isValid()` method.

```ts
if (draft.isValid()) {
  console.log('Purchase agreement is valid')
} else {
  console.log('Errors:', draft.validate().errors)
}
```
</Step>

<Step>
### Render

Let's render the purchase agreement with the data using the `.render()` method.

```ts
import { textRenderer } from '@paradoc/sdk'
import { createFsResolver } from '@paradoc/resolvers'

const renderer = textRenderer()
const resolver = createFsResolver({ root: process.cwd() })

const output = await draft.render({ renderer, resolver, layer: 'markdown' })

console.log(output)
// Purchase Agreement
//
// Alice Johnson agrees to purchase 100 units at $25.00 each.
// Bob Smith agrees to deliver by March 1, 2025.
```
</Step>
