The `@paradoc/sdk` package is an umbrella package for creating, validating, rendering Paradoc artifacts.

# Installation

  </Tab>
  <Tab value="yarn">

  <Tab value="bun">

  </Tab>
</Tabs>



# Package Overview

The SDK is an umbrella package that re-exports functionality from:

| Package | Description |
|---------|-------------|
| `@paradoc/core` | Builders, instances, and lifecycle types for all artifact types |
| `@paradoc/types` | TypeScript type definitions |
| `@paradoc/renderer-pdf` | PDF output |
| `@paradoc/renderer-docx` | DOCX output |
| `@paradoc/renderer-text` | Plain text and markdown output |
| `@paradoc/serialization` | Locale-aware value formatting for rendering |
