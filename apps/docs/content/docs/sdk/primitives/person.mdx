Build and validate person information. The Person primitive captures individual name components and personal details.

# Examples

## Creating persons

```ts
// Object pattern
const person = para.person({
  name: 'John Michael Smith'
})

// Builder pattern
const person = para.person()
  .name('John Michael Smith')
  .title('Dr.')
  .firstName('John')
  .middleName('Michael')
  .lastName('Smith')
  .suffix('Jr.')
  .build()

// Copy and modify
const updated = para.person()
  .from(person)
  .title('Prof.')
  .build()
```

## Parsing external data

```ts
// Parse unknown input (throws on error)
const person = para.person.parse(jsonData)

// Safe parsing (returns result object)
const result = para.person.safeParse(jsonData)
if (result.success) {
  console.log(`Name: ${result.data.name}`)
} else {
  console.error(result.error.message)
}
```

## Using with forms

```ts
const form = para.form({
  name: 'registration',
  fields: {
    applicant: { type: 'person', label: 'Applicant Information' }
  }
})

const filled = form.fill({
  fields: {
    applicant: para.person()
      .name('Jane Doe')
      .firstName('Jane')
      .lastName('Doe')
      .build()
  }
})
```

# API

## Object Pattern

```ts
para.person(input: Person): Person
```

Pass a Person object for validation. Returns the validated Person or throws on error.

```ts
const person = para.person({ name: 'Jane Doe' })
```

# Common Titles

| Title | Usage |
|-------|-------|
| `Mr.` | Adult male |
| `Mrs.` | Married female |
| `Ms.` | Female (marital status neutral) |
| `Miss` | Unmarried female |
| `Dr.` | Doctor (medical or academic) |
| `Prof.` | Professor |
| `Rev.` | Religious leader |

# Validation

The Person primitive validates:

- `name` must be a non-empty string
- All other fields are optional

```ts
// Valid
para.person({ name: 'John Doe' })
para.person({
  name: 'Dr. John Michael Smith Jr.',
  title: 'Dr.',
  firstName: 'John',
  middleName: 'Michael',
  lastName: 'Smith',
  suffix: 'Jr.'
})

// Invalid - throws Error
para.person({})
// Error: Invalid Person: name is required

para.person({ name: '' })
// Error: Invalid Person: name cannot be empty
```
