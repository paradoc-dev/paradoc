# paradoc-cli

Convenience alias for [`@paradoc/cli`](https://www.npmjs.com/package/@paradoc/cli). Installing this package gives you the `para` and `paradoc` commands — same binaries, same behavior, just a shorter unscoped name to type.

## Install

```bash
npm install -g paradoc-cli
# or, without installing:
npx paradoc-cli
```

Equivalent to:

```bash
npm install -g @paradoc/cli
```

## Documentation

Full CLI reference, configuration, and command list: [paradoc.dev/docs/cli](https://paradoc.dev/docs/cli).

For programmatic use of the framework, install [`@paradoc/sdk`](https://www.npmjs.com/package/@paradoc/sdk) directly.
